# Compte Rendu : Analyse et Modélisation Machine Learning

## Introduction

Ce projet porte sur l'analyse et la modélisation prédictive de deux jeux de données publics :  
- La qualité du vin blanc  
- Le défaut de paiement des clients par carte de crédit  

Le but est d'appliquer des méthodes d'apprentissage automatique pour comprendre et prédire les résultats.

## Description des Données

### Qualité du vin blanc

- 4898 échantillons  
- 12 variables (acidité, pH, alcool, etc.)  
- Classification binaire : vin de qualité "bonne" ou "mauvaise"

### Défaut de paiement carte de crédit

- 30000 échantillons  
- 23 variables démographiques et financières  
- Variable cible : défaut de paiement (Oui/Non)  

## Méthodologie

- Nettoyage et préparation des données avec pandas  
- Analyse exploratoire avec visualisations (boxplots, heatmaps)  
- Construction de modèles de classification pour prédire les cibles  
- Évaluation des modèles par métriques pertinentes

## Résultats

- Corrélations significatives identifiées entre certaines variables clés  
- Réseau de neurones artificiel performant pour estimation de la probabilité de défaut

## Conclusion

Les résultats montrent l'efficacité de certains modèles de machine learning pour la prédiction sur ces datasets. Des améliorations passeraient par une validation croisée et un ajustement plus approfondi des hyperparamètres.
